## Reflection
> *(Replace this with your own 150–200 word write-up)*

For this task, I chose to set up profiles on GitHub, LinkedIn, and Kaggle — three platforms that are directly relevant to my career as an engineering student. GitHub is where developers host their code projects, collaborate on open-source software, and showcase their technical work. Even as a first-year student with no major projects yet, having a GitHub profile with a professional README signals to internship recruiters that I am serious about my field. LinkedIn is the world's largest professional networking platform, used by employers, alumni, and industry professionals to connect. I filled in my current education details and made sure my profile looks professional and minimal for now. Kaggle is a data science and machine learning community where students can practice with real datasets and participate in competitions. I chose it because data skills are increasingly relevant across all engineering disciplines. Over the next four years, I plan to use GitHub to host every project I build, LinkedIn to connect with seniors and explore internship opportunities, and Kaggle to gradually improve my data analysis skills. Together, these three platforms will form the core of my professional digital identity.
